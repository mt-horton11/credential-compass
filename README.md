# CredentialCompass

AI-powered clinical license relocation navigator.

## Deploy to Netlify (easiest)

1. Drag this entire folder to https://app.netlify.com/drop
   — OR —
   Push to GitHub and connect the repo to Netlify (it will auto-build)

## Run locally

```bash
npm install
npm run dev
```

Then open http://localhost:5173

## Build for production

```bash
npm install
npm run build
```

Drag the `dist/` folder to Netlify Drop.
